# Code of Conduct

This project uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Standards

Participants should communicate respectfully, assume good intent when possible,
and focus disagreement on facts, design, safety and buyer/consumer impact.

Unacceptable behavior includes harassment, threats, discrimination, publishing
private information, or using the project to fabricate or misrepresent
vehicle title, lien, or odometer data.

## Enforcement

Maintainers may remove comments, reject contributions or restrict
participation when behavior puts contributors, users, operators, or buyers
relying on this data's accuracy at risk.

Report conduct concerns privately to the maintainers through the cloud-itonami
organization.
